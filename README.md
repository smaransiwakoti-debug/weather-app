# Weather App

A clean, responsive weather application that fetches real-time weather data using the Open-Meteo API. Search any city and get current conditions instantly. Built with vanilla JavaScript, no heavy frameworks.

## Features

- Search weather by city name
- Real-time temperature and current conditions
- Fully responsive (works on mobile and desktop)
- Fast and lightweight (vanilla JS, no framework overhead)
- Clean UI powered by Bootstrap 5

## Tech Stack

- HTML5
- CSS3 / Bootstrap 5
- JavaScript (ES6+), fetch API and DOM manipulation
- Open-Meteo API free, no-key weather data

## Getting Started

Clone the repo:

    git clone https://github.com/smaransiwakoti-debug/weather-app.git
    cd weather-app

Then open index.html in your browser, or use the Live Server extension in VS Code.

## What I Learned

- Making API calls with the fetch API and handling async responses
- Dynamically updating the DOM with live data
- Building responsive layouts with Bootstrap 5
